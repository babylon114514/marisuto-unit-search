Q.ユニットクリックした時の詳細画面邪魔なんだけど、ひでしね
A.ダブルクリックで消えます
Q.二重ソートしたいんよ〜

Q.OR検索モード？
A.複数タイプ選択した時に[魔法使い＆騎士]で検索されるのが[魔法使いor騎士]になったりする
Q.右クリ(ロック済み)ユニット固定表示って何だよ(哲学)
A.ユニットリストの項目を右クリックしたら薄水色になる…なるよね？
　　その状態のユニットをどんな検索条件でも必ずリストに表示する為の機能です
Q.STP兄貴お疲れ様です！
A.(STP兄貴じゃ)ないです
Q.DPCって何だよ
A.Damage per cost(要は"DPS"のコスト版)
　　(ATK÷COST)×攻撃回数÷サイズ。スキルとか無視して殴りだけの性能です
